# YouBeenClassed

The internet's AI judgment engine. Submit ideas, receive structured verdicts.

## 🚀 Live Site

**https://youbeenclassed.org**

## Features

- **AI Verdict Engine**: GPT-4 powered analysis
- **10 Categories**: Ideas, Business, Startups, Stories, and more
- **3 Tone Modes**: Professional, Brutal Honesty, Roast
- **Payment Tiers**: Free, Casual ($4.99/mo), Pro ($14.99/mo), Team ($49/mo)
- **Social Features**: Share cards, public feed, leaderboards, appeals

## Tech Stack

- React + TypeScript + Vite + Tailwind CSS
- Clerk (Auth)
- Stripe (Payments)
- OpenAI (AI)
- Vercel (Hosting)
